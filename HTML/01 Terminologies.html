1. Client : Send requests for HTML files
  2. Server : Returns HTML files
  3. Client Server architechture : is a network model that allows applications to communicate and exchange data over a network
  4. GUI : (Graphical User Interface) is a way to interact with a computer or other electronic device using graphical elements
  5. Web Browser : A web browser is a software application that uses HTML to display information from the web to the user
  6. address bar : the familiar text field at the top of a web browser's graphical user interface (GUI) that displays the name or the URL (uniform resource locator) of the current web page
  7. web address : The web address contains information about the location of the webpage. It is also known as the URL (uniform resource locator)
  9. Domain : a property of a document
  10. subdomain : a piece of additional information added to the beginning of a website's domain name
  11. URL : the address of a unique resource on the internet
  12. URI :(Uniform Resource Identifier)
  13. Web page : A web page is a document written in HyperText Markup Language (HTML) that can be displayed in a web browser
  14. Website : A website in HTML is a collection of web pages that are linked together and accessed through a single domain name or URL
  15. Hosting : HTML hosting is a service that allows users to upload and store HTML files on a web server, making their website accessible to the public. It's a key component for static websites, which are websites with fixed content that doesn't change
  16. server space : a storage space for files on a server that can be accessed permanently via the Internet
  17. TCP/IP : a set of rules that allow devices to communicate over the internet.
  18. Protocol : a standard that specifies how data is transmitted between computers
  19. HTTP / HTTPs : HTTP is the foundation of the World Wide Web and is the primary protocol for transferring information across the internet.
  20. SEO : is a practice that uses HTML tags to improve a website's visibility in search engines
  21. Top Domain name SEO : A top-level domain (TLD) is the part of a domain name that comes after the second-level domain (SLD) and is used to determine a website's credibility, trustworthiness, and geographical or topical targeting
  22. What is Text Editor : is a program that allows users to create, evaluate, and edit HTML code, which is the markup for a web page.
  23. File / Folder : A file path describes the location of a file in a web site's folder structure.
  24. Extension : The most common file extensions for HTML (Hypertext Markup Language) files are .html and .htm
